# ライントレーサー作成補助のための光測定実験

## 概要
- フォトトランジスタを光センサー（ライン検知）として利用してライントレーサーを作成する際に
光の反射値のしきい値を求めるための補助測定装置

## 構成
- PythonとArduinoのシリアル通信で光の反射値を1000点ほど測定する
- ライントレースとなる黒線と白線を印刷した紙をフォトトランジスタに約2秒ほど(arduinoのloop関数で1000点の光反射値を測定する時間)交互に
測定する．この1000点をPythonでプロットし、同時に1000点分の平均値を求め、平均値の直線をプロットする．
- CSVファイルも出力されるので、Excel等のツールを用いて自分で外れ値など除外してしきい値を求めることも可能
