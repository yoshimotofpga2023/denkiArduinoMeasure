# PWM駆動によるDCモーターの速度測定

## 概要
- DCモーターを4段階ほどのArduinoからのPWM波形（電圧）で駆動し、DCモーターの10回転にかかる時間を求める．
- 求めた時間から回転速度(rpm)を求めて、PWM値に比例して速度が線形に上がることが確認できる．

## 構成

- DCモーター(ギアモーター)をFET経由でArduinoPWM出力ポートで10回転させる．
- Arduino内で10回転する時間を計測する．
- DCモーターの１回転はリードスイッチを利用して、DCモーターに固定した磁石が通ったときにカウントする．
- リードスイッチを利用するのでチャタリングをソフトウェア的に処理して防止している．
- ４点のPWM値でDCモーター10回転の秒数をシリアル通信でPC(Python)送信する．
- Pythonで4点測定後に横軸PWM値、縦軸モーター速度をプロットする．これはだいたい線形にプロットされる．
そのため、最小二乗法を利用して(pythonライブラリ)近似直線をプロットする．同時に、凡例にて直線の傾きを求める．
プロットと同時にCSVファイルも出力されるので、自身でExcel等のツールでプロットと近似直線を求める作業ができる．
